🎨 Gerador de CSS com IA

Este projeto é um Gerador de CSS utilizando Inteligência Artificial, desenvolvido durante as aulas ao vivo do canal DevClub, ministradas pelo professor Rodolfo Mori.

A aplicação permite que o usuário descreva o estilo desejado e receba automaticamente um código CSS gerado por IA, tornando o desenvolvimento front-end mais rápido e produtivo.

🤖 Como Funciona

O usuário descreve o estilo desejado (ex: botão moderno, layout flexbox, card responsivo).

A aplicação envia a solicitação para a API da Groq.

A IA processa o pedido.

O CSS gerado é exibido em tempo real para uso.
